과거의 어느 날로 떨어진
트레이더들의 생존 대결

1998–2025년 사이 무작위 시점, 정체를 숨긴 종목 7개(주식 4·코인 3). 같은 차트를 보고 누가 더 버는지 겨룹니다. 날짜와 실명은 정산 때 공개됩니다.

데이터 안내 — 종목의 큰 흐름(폭락·랠리)은 실제 역사적 가격 지점을 따르되, 일 단위 시세는 실제 변동성 수준으로 재구성한 근사치입니다. 종목 가명은 판마다 무작위로 바뀌며 실명은 정산 때 공개됩니다. 미국 주식 환율 1,300원 고정, 코인은 소수점 수량을 1주로 환산(단위는 종목 설명에 표기)하며 주말 거래는 생략합니다. 수수료 없음.

대결 서버 연결하기

대결 모드는 참가자들의 데이터를 주고받을 서버가 필요합니다. 구글 Firebase 무료 플랜이면 충분하며, 설정은 한 번만 하면 됩니다 (약 10분).

1단계 — Firebase 프로젝트 만들기

01console.firebase.google.com 접속 → 구글 로그인 → 프로젝트 만들기 (이름 자유, 애널리틱스는 꺼도 됨)
02왼쪽 메뉴 Databases & Storage → Realtime Database (안 보이면 상단 검색창에 Realtime Database 입력, 또는 console.firebase.google.com/project/_/database 직접 접속) → 데이터베이스 만들기 → 위치 asia-southeast1 → 테스트 모드로 시작
03테스트 모드는 30일 후 잠기므로, Database의 규칙 탭에서 아래로 교체 후 게시:
{"rules":{"rooms":{".read":true,".write":true}}}
⚠ 이 규칙은 누구나 rooms에 쓸 수 있는 공개 설정입니다. 이 게임 전용 프로젝트로만 쓰세요.
04⚙ 프로젝트 설정 → 내 앱 → 웹(</>) 앱 추가 → 앱 등록 → 화면에 나오는 firebaseConfig = { ... } 코드 전체를 복사

2단계 — 아래에 붙여넣기

참가자 설정 생략하기 — 이 설정은 방장 1명만 하면 됩니다. 파일 코드의 EMBEDDED_FB_CONFIG = null 자리에 config를 붙여넣어 배포하면, 파일을 받는 사람은 아무 설정 없이 바로 참가합니다.

3단계 — 친구가 링크로 접속하려면 (호스팅)

이 HTML 파일이 대장 폰에만 있으면 남이 열 수 없습니다. 무료 호스팅에 올리면 참여 링크 하나로 누구나 접속합니다.

ANetlify Drop (가장 쉬움): app.netlify.com/drop에 이 파일을 드래그 → 즉시 주소 발급. 파일명은 index.html로 바꿔서 올리세요.
BGitHub Pages: 저장소에 index.html로 올리고 Settings → Pages 활성화.
호스팅 없이도 각자 이 파일을 받아서 열고, 방 코드를 직접 입력하면 대결이 됩니다. 링크 자동입장만 호스팅이 필요합니다.

실제 화면 위에서 기능을 하나씩 비추며 설명합니다 · 아래는 글로 보는 참고용 정리

🎮 한 판의 흐름

01방 만들기 — 자본금·기간을 숫자로 정하고, 원하면 🔥강철의 심장(역사적 폭락·버블이 반드시 포함), ⚡레버리지, 📉공매도를 켭니다. 혼자 연습도 가능합니다.
02친구 초대 — 방 코드 4자리를 부르거나, 상단의 🔗링크(누르면 복사)를 공유하면 코드가 자동 입력된 채로 입장합니다.
03하루 거래 — 과거 20일 차트와 📰당시 소문을 참고해 사고팝니다. 다 했으면 장 마감 버튼.
04전원 대기 — 모두가 마감해야 다음 날이 열립니다. 기다리는 동안 마감자끼리 💬채팅으로 눈치 싸움을 하세요 (수익률은 서로 안 보입니다).
05개장 브리핑 — 새 날이 열리면 전 종목 등락률이 5초간 공개됩니다. 마지막 날 마감 후 정산에서 순위·실명·실제 날짜·사건이 한꺼번에 공개됩니다.

🛒 주문하는 법

비율수량 대신 슬라이더(1~100%)로 주문합니다. 매수는 매수 여력의 %, 매도는 보유량의 %. 버튼에 실제 체결 주수가 미리 표시됩니다.
배수⚡레버리지 방에서는 이번 거래 배수(1x~한도)를 골라야 빚투가 됩니다. 기본은 항상 1x — 실수로 풀레버리지 걸릴 일은 없습니다. 3x 선택 = 순자산의 3배까지 매수(모자란 돈은 차입, 현금이 음수로 표시).
숏📉공매도 방에서는 보유가 0이어도 매도가 됩니다(버튼에 "매도·숏"으로 표시). 빌린 주식을 판 돈이 현금에 들어오고, 가격이 내리면 이익입니다. 되사려면 매수를 누르면 됩니다.

📊 화면 읽는 법

차트회색 선 = 과거 20일(참고용), 색 선 = 진행 구간. 빨강=상승, 파랑=하락 (한국 증권 앱 관례).
소문📰당시 소문 = 그 시기의 실제 호재·악재를 실명·연도 없이 흐릿하게 쓴 것. 소문으로 시대와 정체를 추리하는 게 이 게임의 묘미입니다.
랭킹상단 띠는 순위만 보여줍니다 (👑=1위, 밝은 칩=나). 남의 수익률은 정산 때까지 비밀.
노출레버리지 방의 "노출 2.4x / 한도 3x · 차입 ₩…" = 지금 순자산 대비 몇 배를 태우고 있는지. 빨간색이면 위험 구간입니다.

💀 파산 규칙 (레버리지·숏 방만 해당)

하루가 열렸을 때 순자산(현금+평가액)이 0 이하면 전량 강제 청산됩니다. 남은 마이너스는 부채로 확정되어 정산에 -100%를 넘는 손실(예: -120%)로 기록됩니다. 3배 풀매수는 하루 -34%면 파산, 숏은 급등하면 손실이 원금을 넘을 수 있습니다. 1배 현물만 쓰면 파산은 없습니다.

🧭 알아두면 좋은 것

복구앱이 꺼지거나 튕겨도 홈의 "진행 중이던 방으로 복귀"로 재입장하면 지갑·보유가 그대로 복원됩니다.
공정모든 참가자는 완전히 같은 시세·종목·기간을 봅니다 (같은 시드에서 생성). 가명은 판마다 무작위로 바뀝니다.
지뢰종목 중에는 상장폐지급 붕괴를 겪은 코인도 섞여 있습니다. "왜 이렇게 싸지?" 싶으면 이유가 있는 겁니다.
데이터 안내 — 큰 흐름(폭락·랠리)은 실제 역사적 가격 지점을 따르지만, 하루하루 시세는 각 종목의 실제 변동성 수준으로 수학적으로 재구성한 근사치입니다. 실제 과거 종가와 다를 수 있으며, 투자 조언이 아닌 게임입니다.

대결 규칙 정하기

시간이 끝나면 자동으로 장 마감됩니다. 기기별 몇 초 오차 가능 · 완전히 꺼진 기기는 강퇴로 정리하세요.
강철의 심장: 리먼 파산, 코인 광풍, 코로나 폭락 같은 역사적 급변 사건이 반드시 진행 기간 안에 포함됩니다. 어떤 사건인지는 정산 때 공개.
순자산의 N배까지 매수 가능(부족분 차입). 하루가 열렸을 때 순자산이 0 이하면 전량 청산·파산 💀 이자는 없습니다.
보유 0에서 매도하면 하락에 베팅하는 숏 포지션이 열립니다. 총 노출(롱+숏)은 순자산×배수까지. 주가가 오르면 숏은 손실이며 순자산 0 이하 시 청산·파산합니다.
